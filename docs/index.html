基礎篇¶

在這個部分，我們介紹程式設計典範 (paradigm) 中的指令式程式設計 (imperative programming)，包括變數 (variable)、型別 (type)、運算子 (operator)、控制流程 (control flow) 等，也會介紹數種容器 (container)，接著，介紹 struct 和 enum 等複合型別。我們會將程式碼進一步組織成函式 (function)，再將其再包裝成模組 (module) 和套件 (package)，這也是大部分程式設計初學者所接觸的程式設計典範。在學完這個部分後，讀者就可以自行撰寫基本的 Rust 程式，試著用 Rust 解決自己的問題。
